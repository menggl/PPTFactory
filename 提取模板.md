从《1.2 安全生产方针政策.pptx》第五页开始到倒数第二页截止，都是内容页，将这些内容页都抽离出对应的模板文件和对应的策略类风格文件，抽离出的模板都统一放到master_template.pptx文件中
master_template.pptx文件里面的水印都去掉（使用底层遍历xml的方式），包括水印使用的文本框：
“Evaluation only.”
“Created with Aspose.Slides for Java 25.11.”
“Copyright 2004-2025 Aspose Pty Ltd.”
master_template.pptx文件中的文本框中的文字全部修改成重复的“模板文字模板文字模板文字”，保持文本框中的字数与修改文本之前的文本框中的字数相同（字数多了截取，字数少了重复）、文字大小相同、字体样式相同，字体对齐方式相同
master_template.pptx文件中的图片都统一改成有带有“No Image“文字标识的图片，顶部图标除外


根据master_template.pptx文件抽离出master_template_metadata.json，该json文件保存提取出的每个模板页的结构特征
每个模板页自动提取出下面的信息：
① 元素数量
文本框数量（如：1 个标题 + 3 个正文）
图片占位符数量
图表占位符数量
② 布局类型
一列
两列
3 图网格
全屏图片
标题+正文
标题+两侧图文
这些可以通过解析 PPTX XML 自动分析。
③ 占位符类型
比如：
标题 placeholder
body placeholder
picture placeholder
content placeholder
④ 尺寸、位置（bounding boxes）
可判断这是：
左图右文
上标题下三栏
全屏图
图文混排
……
⑤ 模板用途标签（可自动打标签）
你可以自动推断：
cover（封面）
catalog（目录）
content（正文）
summary（总结）
chart（数据展示）
